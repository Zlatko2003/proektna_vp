# DevQ&A - Платформа за прашања и одговори за програмери

Bеб апликација каде програмери можат да поставуваат прашања, да споделуваат знаење и да добиваат одговори од заедницата. Направена со Angular, Node.js, Express и MongoDB.

## Функционалности

- Автентикација на корисници со три улоги: гостин, регистриран корисник и администратор
- Креирање, прегледување, уредување и бришење на прашања
- Објавување одговори и коментари
- Гласање за одговори
- Организација на прашања преку ознаки
- Пребарување и филтрирање на прашања
- Администраторски панел со статистики
- Docker поддршка за лесно поставување

## Структура на проектот

### mockups
Ја содржи статичната HTML верзија на апликацијата. Ова се само визуелни маски на екраните без функционална логика. Користени се за приказ на изгледот на апликацијата пред имплементација на динамичките делови. Некои од маските се во некој елементи малку различни кои претрпиле промена за време на имплементацијата на динамичките делови.

### backend
Содржи Node.js сервер со Express. Овој дел ја реализира backend логиката на апликацијата, комуникацијата со MongoDB базата на податоци преку REST API, автентикацијата со JWT и документацијата на API (Swagger). Содржи модели, контролери, рути и middleware.

### frontend
Содржи Angular апликација која претставува клиентска страна (SPA). Овде се наоѓаат компонентите, сервисите за комуникација со backend-от, pipes, модели на податоци и рутирање. Овозможува динамички приказ на податоците и интеракција со корисникот.
